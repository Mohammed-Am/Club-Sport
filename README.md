**Club Sport** is a Java and Java Swing application designed to manage a sports club. The application allows you to manage gym entry times and client subscriptions easily.

## Features
- Add and manage gym entry times.
- Manage client subscriptions.
- User-friendly interface built with Java Swing.
  
## Technologies Used
- **Java**
- **Java Swing**

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mohammed-Am/Club-Sport.git
2. Navigate to the project directory:
   ```bash
   cd Club-Sport
3. Run the application:
   - If you're using an IDE like NetBeans or Eclipse, you can open and run the project directly.
   - Or compile and run the project from the terminal:
   ```bash
   javac src/clubsport/*.java
   java src/clubsport/ClubSport

## Usage
- **Add the gym entry times for clients.**
- **Manage and update client subscriptions.**   
- **Monitor the club's overall activity.**
